# Use cases

### 1. Создание задачи

<table>
  <tr> 
    <th>Заголовок</th> 
    <td>Создание задачи</td> 
  </tr> 
  <tr> 
    <th>Участники</th> 
    <td>Пользователь трекера</td> 
  </tr> 
  <tr> 
    <th>Предусловия</th> 
    <td>Авторизация в системе, наличие прав на создание задач</td> 
  </tr>
  <tr> 
    <th>Основной сценарий</th> 
    <td>
      <ol>
        <li>Пользователь создает задачу</li>
        <li>Пользователь вводит название, описание, делайн, добавляет теги</li>
        <li>Сервис задач проверяет данные на корректность</li>
        <li>Сервис задач сохраняет задачу</li>
      </ol>
  </td> 
  <tr>
    <th>Постусловия</th> 
    <td> 
      Задача отображается в списке задач, корректно отображаются заголовок, текст, теги и дедлайн задачи
    </td> 
  </tr> 
  <tr> 
    <th>Исключения</th> 
    <td>
      <ul>
        <ol>
          <li>Сервис задач не может сохранить задачу из-за ошибки</li>
        </ol>
        <li>Сервис задач возвращает ответ на запрос с информацией об ошибке при наличии сведений</li>
      </ul>
    </td> 
  </tr> 
</table>

### 2. Постановка задачи

<table>
  <tr> 
    <th>Заголовок</th> 
    <td>Поиск задачи по ключевым словам, фильтрам и тегам</td> 
  </tr> 
  <tr> 
    <th>Участники</th> 
    <td>Пользователь трекера</td> 
  </tr> 
  <tr> 
    <th>Предусловия</th> 
    <td>Авторизация в системе, наличие прав на просмотр задач</td> 
  </tr> 
  <tr> 
    <th>Основной сценарий</th> 
    <td>
      <ol>
        <li>Пользователь вводит поисковой запрос с ключевыми словами и фильтрами</li>
        <li></li>
        <li>Сервис задач возвращает результат поиска через API Gateway</li>
      </ol>
  </td> 
  <tr>
  <th>Постусловия</th> 
  <td> 
    В результатах поиска отображаются все задачи, удовлетворяющие условиям поиска, и не отображаются задачи, им неудовлетворяющие
  </td> 
  </tr>
  <tr> 
    <th>Альтернативные сценарии</th> 
    <td>
      <ol>
        <li>Сервис задач не находит ни одной задачи, удовлетворяющей условиям поиска</li>
        <li>Сервис задач возвращает ответ на запрос с текстом "Задач по указанным критериям не найдено"</li>
      </ol>
    </td> 
  </tr>
  <tr> 
    <th>Исключения</th> 
    <td>
      <ol>
        <li>Сервис задач не может обработать запрос</li>
        <li>Сервис задач возвращает ответ на запрос с информацией об ошибке при наличии сведений</li>
      </ol>
    </td> 
  </tr>
</table>
